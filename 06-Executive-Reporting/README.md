## Incident Response & Tabletop Exercise Program

Developed a comprehensive enterprise incident response program that establishes standardized procedures for detecting, responding to, containing, eradicating, and recovering from cybersecurity incidents. The program includes executive governance, communication protocols, severity classifications, tabletop exercise planning, performance metrics, and continuous improvement processes aligned with industry best practices.

### Key Contributions
Designed an enterprise incident response framework
Developed incident severity classification and escalation procedures
Created cross-functional roles and responsibilities
Established a recurring tabletop exercise program
Defined executive KPIs for incident response performance
Implemented a continuous improvement process through lessons learned
